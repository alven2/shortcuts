-configurar usuario local
git config --global user.name "alven"
git config -- global user.email "ppokevaz@gmail.com"

-creando repositorio (path de projecto)
git init

-verificar el estado de repo
git status

-seleccionar todos los archivos del repo para precommitear y ser detectado
git add .

-para commit (el -m es mensaje y es obli)
git commit -m "hice este cambio"

-para ver el historial de commits
git log
